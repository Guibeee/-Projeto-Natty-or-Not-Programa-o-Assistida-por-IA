# -Projeto-Natty-or-Not-Programa-o-Assistida-por-IA
Este projeto foi desenvolvido como parte do Lab Natty or Not da DIO, com o objetivo de explorar como as IAs Generativas podem auxiliar programadores na criação, otimização e documentação de código. A proposta é demonstrar como a colaboração entre humano e máquina pode gerar resultados eficientes e criativos no desenvolvimento de software.
